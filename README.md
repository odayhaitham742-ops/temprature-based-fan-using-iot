# temprature-based-fan-using-iot
A low-cost cooling controller using an ESP32 manages two 12 V fans based on DS18B20 readings and three thresholds (30/40/55 °C). At the highest level it triggers Blynk and Gmail alerts. The system includes a 12 V-to-5 V regulator plus an NPN level shifter. Hardware tests and simulation verified operation.”
